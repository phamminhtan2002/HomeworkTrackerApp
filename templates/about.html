Features

The homepage of this web application displays homework items being tracked within a table. Each row contains:
  • Course ID
  • Homework type
  • A short description
  • Deadline
  • Edit button
  • Delete button
The navigation buttons at the top of the page allow the user to view the homepage, this about section, and a page to create a new homework item to track.
